# Automobile offers

Automobile offers is a web app for selling automobiles.It is based on React.JS and soft-uni server API.

## Installation


starting the server:

cd server

node server


starting React app

in new terminal

cd client

npm start

```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
